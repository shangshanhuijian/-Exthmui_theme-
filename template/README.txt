主题模板

这是一个 exTHmUI 的主题模板
文件结构：
    assets/
        backgrounds       : 壁纸和锁屏壁纸
        fonts             : 字体
        media             : 开机动画
        sounds            : 声音
        previews          : 预览图
    res/                  : 主题的各种资源
    overlay               : 叠加层
    AndroidManifest.xml

具体制作方法请参考各个文件夹下的说明，以及 res/values/ 下的文件和 res/xml/theme_data.xml
