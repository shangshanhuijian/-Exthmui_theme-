Theme Template

This is exTHmUI Theme Template.

File Structure:
    assets/
        backgrounds       : Wallpaper and LockScreen Wallpaper.
        fonts             : System Fonts.
        media             : bootanimation.
        sounds            : System Sounds.
        previews          : Theme Previews.
    res/                  : Theme Resources.
    overlay               : Theme Overlay.
    AndroidManifest.xml

For detailed production methods, please refer to the files under "/res/values/" and "/res/xml/theme_data.xml"
